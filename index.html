006 — Fraud Lab
Behavioral Intelligence Lab

12+ specialized tools built internally at VPT — detecting, mapping, and exposing affiliate fraud, coupon manipulation, SERP hijacking, and extension behavioral anomalies. All automated. All evidence-backed.

LAB · 01

Clickjacking Detection Framework

Collects 10–20 pages of URLs per keyword, visits each domain, simulates clicks, detects hidden redirections and notification abuse traps. Built from AK_Clickjacking_Wpn.

LAB · 02

Inline Adware Hunting System

Searches adware keywords across engines, scrapes sponsored links, visits each landing page, checks keyword matches — full screenshot evidence output. Source: Adware Hunting.

LAB · 03

Coupon Site Scraper + Network Mapper

Scrapes all brand listings from coupon sites, follows every redirect to expose which affiliate network is running behind each coupon. Source: Couponsite Scrapping + Coupon Site Crawler.

LAB · 04

Web Extension Behavior Monitor

Tests browser extensions across brand sites to detect injected cashback popups, discount overlays, and click-hijacking during active sessions. Source: Web_Extension_Scrapping + ExtensionScraper.

LAB · 05

SERP Modification Checker

Detects when browser extensions modify Google/Bing search results — injecting ads, swapping links, or altering organic rankings. Source: Check Serp Modification by Ext.

LAB · 06

SERP Link Extractor

Extracts all links from search engine result pages for a given keyword — feeds directly into the adware and redirect detection pipeline. Source: Link_Extractor_From_SERP.

LAB · 07

Typosquatting / Domain Redirect Tracker

Opens typo and tracking domains one by one, follows every redirect chain, logs each hop until the final affiliate destination is confirmed. Source: Typo Sqotting.

LAB · 08

Extension Popup Injector Detector

Detects when extensions inject popup overlays — cashback prompts, discount banners, notification traps — into brand pages during live sessions. Source: Ext popup Code.

LAB · 09

Unique Advertiser ID Analyzer

Groups and ranks advertiser/merchant IDs being targeted across networks and brands — maps the full attack surface per advertiser. Source: unique advertiser code.

LAB · 10

VM Wise Adware Tracker

Tracks which adware extensions are installed across which VMs and browsers — feeds the Dashboard VM Adware tab via live API. Source: VmWiseAdwareAdd.

LAB · 11

Honey Notification Trap

Monitors and traps abusive notification push patterns from adware extensions — identifies notification abuse campaigns in real time. Source: Honey Notification.

LAB · 12

Dashboard Data Server

Backend data pipeline that feeds the Dashboard Analytics compliance dashboard — scrapes, processes, and structures findings for reporting. Source: Dashboard_data_server.
